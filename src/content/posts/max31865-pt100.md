# STM32F103C8T6 + MAX31865 读取PT100温度

今天来教大家如何使用 STM32F103C8T6 通过 MAX31865 模块读取 PT100 铂电阻温度传感器~

## 准备工作 {#准备工作}

### 1. 所需硬件 {#所需硬件}

- **STM32F103C8T6 开发板** - 核心控制单元，最小系统板即可
- **MAX31865 模块** - PT100专用温度转换芯片，推荐带参考电阻的模块
- **PT100 铂电阻传感器** - 注意选择合适的封装（如三线制）
- **杜邦线若干** - 建议使用优质杜邦线减少干扰
- **面包板** - 便于临时接线测试

### 2. 硬件连接说明

MAX31865 支持 2/3/4 线制连接，推荐使用 **3线制** 以消除导线电阻影响。

**接线表：**

| MAX31865 引脚 | 功能说明 | STM32F103 (PA引脚) | STM32引脚类型 |
|--------------|---------|-------------------|--------------|
| VCC | 电源输入 (3.3V) | 3.3V | 电源 |
| GND | 地线 | GND | 接地 |
| SDO (SO) | SPI数据输出 | PA0 | 输入 |
| CS (SS) | 片选信号 | PA1 | 输出 |
| SCLK (SCK) | SPI时钟 | PA2 | 输出 |
| SDI (SI) | SPI数据输入 | PA3 | 输出 |
| DRDY | 数据就绪信号（可选） | PA15 | 输入 |
| RTD+ | PT100正极 | PT100红/白线 | - |
| RTD- | PT100负极 | PT100黑线 | - |
| REF+ | 参考电阻端（模块内部已接） | - | - |
| REF- | 参考电阻端（模块内部已接） | - | - |

**PT100 三线制接线：**

```
PT100传感器        MAX31865模块
   │                  │
   ├─ 红色线 ──────► RTD+
   ├─ 白色线 ──────► RTD-
   └─ 黑色线 ──────► 与RTD+短接（补偿导线电阻）
```

> **注意**：如果使用四线制PT100，需要将两根补偿线分别连接到 REF+ 和 REF-。

## 工作原理 {#代码详解}

### MAX31865 简介

MAX31865 是 Maxim 公司生产的专用 PT100/PT1000 铂电阻温度传感器数字转换芯片，内部集成了高精度ADC和校正电路，直接输出数字信号。

**主要特性：**

- **内置24位ADC** - 高精度模数转换，无需外部ADC芯片，分辨率可达0.003°C
- **支持2/3/4线制** - 可根据需求选择不同的接线方式，3线制可有效消除导线电阻影响
- **故障检测** - 内置RTD开路和短路检测功能，便于诊断传感器故障
- **SPI接口** - 兼容标准SPI通信协议，最高支持5MHz时钟
- **自动转换** - 可配置为连续自动温度转换模式，无需手动触发
- **宽温度范围** - 支持-200°C至+850°C测量范围（取决于传感器）

**寄存器说明：**

| 寄存器地址 | 名称 | 说明 |
|-----------|------|------|
| 0x00 | CONFIG | 配置寄存器，设置转换模式、滤波、激励电流等 |
| 0x01 | RTD MSB | RTD电阻值高8位（15-8位） |
| 0x02 | RTD LSB | RTD电阻值低8位（7-0位），最低位为0 |
| 0x03 | HIGH FAULT THRESHOLD MSB | 故障阈值上限高字节 |
| 0x04 | HIGH FAULT THRESHOLD LSB | 故障阈值上限低字节 |
| 0x05 | LOW FAULT THRESHOLD MSB | 故障阈值下限高字节 |
| 0x06 | LOW FAULT THRESHOLD LSB | 故障阈值下限低字节 |
| 0x07 | FAULT STATUS | 故障状态寄存器，指示各类故障 |

**CONFIG寄存器位定义（地址0x00）：**

| 位 | 名称 | 说明 |
|---|------|------|
| BIT 7 (VBIAS) | 激励电流开关 | 1=开启，0=关闭 |
| BIT 6 (AUTO) | 自动转换模式 | 1=连续转换，0=单次转换 |
| BIT 5 (3WIRE) | 3线制模式 | 1=启用，0=2/4线制 |
| BIT 4-3 | 预留 | 保留位 |
| BIT 2 (FAULT) | 故障检测 | 1=启用，0=禁用 |
| BIT 1-0 | FILTER | 滤波选择 | 00=60Hz, 01=50Hz, 10=60Hz, 11=50Hz |

### PT100 温度传感器原理

PT100 是一种铂电阻温度传感器，其电阻值随温度变化呈近似线性关系。

**基本参数：**
- **R0**：0°C时的电阻值 = 100Ω
- **温度系数**：α = 0.00385 Ω/Ω/°C
- **测量范围**：-200°C ~ +850°C

**电阻-温度关系：**

PT100的阻值随温度变化遵循 **Callendar-Van Dusen** 公式：

**正温区 (0°C ≤ T ≤ 850°C)：**
$$R_t = R_0(1 + aT + bT^2)$$

其中：
- a = 3.9083 × 10⁻³ °C⁻¹
- b = -5.775 × 10⁻⁷ °C⁻²

**负温区 (-200°C ≤ T < 0°C)：**
$$R_t = R_0(1 + aT + bT^2 + c(T-100)T^3)$$

其中：
- c = -4.183 × 10⁻¹² °C⁻⁴

### 温度计算原理

MAX31865内部通过参考电阻将PT100的电阻转换为数字信号：

$$R_t = \frac{ADC值}{32768} \times R_{REF}$$

其中：
- **ADC值**：从寄存器读取的15位数据（0-32767）
- **R_REF**：参考电阻，常见值为400Ω或430Ω

**温度计算步骤：**

1. 从MAX31865读取RTD数据寄存器（0x01和0x02）
2. 将16位数据右移1位，得到15位ADC值
3. 根据参考电阻计算PT100实际电阻值Rt
4. 使用Callendar-Van Dusen公式计算温度
5. 对于负温度，使用多项式拟合公式

### max31865.c 实现文件

```C
#include "max31865.h"

/************************** 底层引脚操作 **************************/
#define CS_LOW()     GPIO_ResetBits(MAX31865_PORT, MAX31865_PIN_CS)
#define CS_HIGH()   GPIO_SetBits(MAX31865_PORT, MAX31865_PIN_CS)
#define SCLK_LOW()  GPIO_ResetBits(MAX31865_PORT, MAX31865_PIN_SCLK)
#define SCLK_HIGH() GPIO_SetBits(MAX31865_PORT, MAX31865_PIN_SCLK)
#define SDI_LOW()   GPIO_ResetBits(MAX31865_PORT, MAX31865_PIN_SDI)
#define SDI_HIGH()  GPIO_SetBits(MAX31865_PORT, MAX31865_PIN_SDI)
#define SDO_READ()  GPIO_ReadInputDataBit(MAX31865_PORT, MAX31865_PIN_SDO)

/************************** 寄存器定义 **************************/
#define MAX31865_REG_CONFIG        0x00
#define MAX31865_REG_RTD_MSB       0x01
#define MAX31865_REG_RTD_LSB       0x02
#define MAX31865_REG_FAULT         0x07
#define MAX31865_WRITE_MASK        0x80    // 写操作位

#define CFG_VBIAS_ON               (1<<7)  // 开启激励电流
#define CFG_AUTO_CONVERT           (1<<6)  // 自动转换模式
#define CFG_3WIRE_MODE             (1<<5)  // 三线制模式
#define CFG_FAULT_DETECT           (1<<2)  // 故障检测使能
#define CFG_FILTER_50HZ            (1<<0)  // 50Hz滤波

/************************** SPI读写函数 **************************/
static void MAX31865_WriteByte(uint8_t data)
{
    for(uint8_t i = 0; i < 8; i++)
    {
        SCLK_LOW();
        if(data & 0x80) SDI_HIGH();
        else SDI_LOW();
        SCLK_HIGH();
        data <<= 1;
    }
}

static uint8_t MAX31865_ReadByte(void)
{
    uint8_t data = 0;
    for(uint8_t i = 0; i < 8; i++)
    {
        SCLK_LOW();
        data <<= 1;
        SCLK_HIGH();
        if(SDO_READ()) data |= 0x01;
    }
    return data;
}

/************************** 寄存器操作 **************************/
static MAX31865_StatusTypeDef MAX31865_WriteReg(uint8_t addr, uint8_t data)
{
    CS_LOW();
    MAX31865_WriteByte(addr | MAX31865_WRITE_MASK);
    MAX31865_WriteByte(data);
    CS_HIGH();
    return MAX31865_OK;
}

static MAX31865_StatusTypeDef MAX31865_ReadReg(uint8_t addr, uint8_t *data)
{
    if(data == NULL) return MAX31865_PARAM_ERR;
    CS_LOW();
    MAX31865_WriteByte(addr);
    *data = MAX31865_ReadByte();
    CS_HIGH();
    return MAX31865_OK;
}

/************************** GPIO初始化 **************************/
static void MAX31865_GPIO_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStructure;
    
    // 使能GPIOA时钟
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA, ENABLE);
    
    // SDO - 输入
    GPIO_InitStructure.GPIO_Pin = MAX31865_PIN_SDO;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_IN_FLOATING;
    GPIO_Init(MAX31865_PORT, &GPIO_InitStructure);
    
    // CS, SCLK, SDI - 输出
    GPIO_InitStructure.GPIO_Pin = MAX31865_PIN_CS | MAX31865_PIN_SCLK | MAX31865_PIN_SDI;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(MAX31865_PORT, &GPIO_InitStructure);
    
    // DRDY - 输入（可选）
    GPIO_InitStructure.GPIO_Pin = MAX31865_PIN_DRDY;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_IN_FLOATING;
    GPIO_Init(MAX31865_PORT, &GPIO_InitStructure);
    
    // 默认状态
    CS_HIGH();
    SCLK_HIGH();
}

/************************** 初始化 **************************/
MAX31865_StatusTypeDef MAX31865_Init(void)
{
    MAX31865_StatusTypeDef status;
    uint8_t cfgValue = 0;

    // GPIO初始化
    MAX31865_GPIO_Init();
    
    // 软件复位
    status = MAX31865_SoftReset();
    if(status != MAX31865_OK) return status;

    // 组装配置寄存器值
    cfgValue |= CFG_VBIAS_ON;        // 开启激励电流
    cfgValue |= CFG_AUTO_CONVERT;    // 自动转换模式
    cfgValue |= CFG_FAULT_DETECT;    // 使能故障检测
    
    #if RTD_WIRE_3LINE
    cfgValue |= CFG_3WIRE_MODE;      // 三线制模式
    #endif
    
    #if RTD_FILTER_50HZ
    cfgValue |= CFG_FILTER_50HZ;     // 50Hz滤波
    #endif

    return MAX31865_WriteReg(MAX31865_REG_CONFIG, cfgValue);
}

/************************** 获取温度 **************************/
MAX31865_StatusTypeDef MAX31865_GetTemperature(float *temp)
{
    if(temp == NULL) return MAX31865_PARAM_ERR;

    uint8_t regH, regL, fault;
    uint16_t adcData;
    float Rt, temperature;

    // 读取故障状态
    fault = MAX31865_ReadFaultStatus();
    if(fault & 0x04) return MAX31865_RTD_OPEN;   // RTD开路
    if(fault & 0x02) return MAX31865_RTD_SHORT;  // RTD短路

    // 读取RTD数据寄存器
    MAX31865_ReadReg(MAX31865_REG_RTD_MSB, &regH);
    MAX31865_ReadReg(MAX31865_REG_RTD_LSB, &regL);

    // 组合16位数据，右移1位得到15位有效数据
    adcData = ((uint16_t)regH << 8 | regL) >> 1;
    
    // 检查数据有效性
    if(adcData == 0 || adcData >= 32767) 
        return MAX31865_ERROR;

    // 计算PT100电阻值
    Rt = (float)adcData / 32768.0f * RTD_RREF_VALUE;

    // 使用Callendar-Van Dusen公式计算温度
    const float a = 3.9083e-3f;
    const float b = -5.775e-7f;

    float Z1 = -a;
    float Z2 = a * a - 4 * b;
    float Z3 = 4 * b / RTD_R0_VALUE;
    float Z4 = 2 * b;
    
    temperature = Z2 + Z3 * Rt;
    temperature = (sqrtf(temperature) + Z1) / Z4;

    // 正温度直接返回
    if(temperature >= 0.0f)
    {
        *temp = temperature;
        return MAX31865_OK;
    }

    // 负温度使用IEC751多项式拟合
    float rpoly = Rt;
    temperature = -242.02f;
    temperature += 2.2228f * rpoly;
    rpoly *= Rt;
    temperature += 2.5859e-3f * rpoly;
    rpoly *= Rt;
    temperature -= 4.8260e-6f * rpoly;

    *temp = temperature;
    return MAX31865_OK;
}

/************************** 软件复位 **************************/
MAX31865_StatusTypeDef MAX31865_SoftReset(void)
{
    return MAX31865_WriteReg(MAX31865_REG_CONFIG, 0x80);
}

/************************** 读取故障状态 **************************/
uint8_t MAX31865_ReadFaultStatus(void)
{
    uint8_t fault = 0;
    MAX31865_ReadReg(MAX31865_REG_FAULT, &fault);
    return fault;
}

/************************** 设置转换模式 **************************/
MAX31865_StatusTypeDef MAX31865_SetConvertMode(uint8_t autoMode)
{
    uint8_t cfg;
    MAX31865_ReadReg(MAX31865_REG_CONFIG, &cfg);
    
    if(autoMode)
        cfg |= CFG_AUTO_CONVERT;
    else
        cfg &= ~CFG_AUTO_CONVERT;
    
    return MAX31865_WriteReg(MAX31865_REG_CONFIG, cfg);
}
```

### main.c 主函数示例

```C
#include "max31865.h"
#include "stdio.h"

// 假设已经实现了串口打印函数
extern void USART_SendString(char* str);
extern void USART_SendFloat(float value);

int main(void)
{
    float temperature;
    MAX31865_StatusTypeDef status;
    char buffer[50];

    // 初始化MAX31865
    status = MAX31865_Init();
    if(status != MAX31865_OK)
    {
        USART_SendString("MAX31865初始化失败！\r\n");
        while(1);  // 初始化失败，死循环
    }
    
    USART_SendString("MAX31865初始化成功！\r\n");

    while(1)
    {
        // 获取温度
        status = MAX31865_GetTemperature(&temperature);
        
        if(status == MAX31865_OK)
        {
            // 温度有效，输出结果
            sprintf(buffer, "温度: %.2f°C\r\n", temperature);
            USART_SendString(buffer);
        }
        else
        {
            // 处理故障
            switch(status)
            {
                case MAX31865_RTD_OPEN:
                    USART_SendString("错误: RTD开路！\r\n");
                    break;
                case MAX31865_RTD_SHORT:
                    USART_SendString("错误: RTD短路！\r\n");
                    break;
                default:
                    USART_SendString("错误: 读取失败！\r\n");
                    break;
            }
        }

        // 延时1秒
        for(uint32_t i = 0; i < 7200000; i++);
    }
}
```

### MAX31865.c

```C
#include "MAX31865.h"

/************************** 初始化 **************************/
void MAX31865_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct = {0};

    // 使能GPIOA时钟
    __HAL_RCC_GPIOA_CLK_ENABLE();

    // 配置CS, SCLK, SDI为输出
    GPIO_InitStruct.Pin = MAX31865_CS | MAX31865_SCLK | MAX31865_SDI;
    GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;
    GPIO_InitStruct.Pull = GPIO_NOPULL;
    GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_HIGH;
    HAL_GPIO_Init(MAX31865_CONTROL_PORT, &GPIO_InitStruct);

    // 配置SDO为输入
    GPIO_InitStruct.Pin = MAX31865_SDO;
    GPIO_InitStruct.Mode = GPIO_MODE_INPUT;
    HAL_GPIO_Init(MAX31865_CONTROL_PORT, &GPIO_InitStruct);

    // 默认状态
    MAX31865_CS_SET;
    MAX31865_SCLK_SET;
    
    // 初始化配置
    MAX31865_Write(0x80, 0xD1);  // 开启自动转换、三线制、50Hz滤波
}

/************************** 写寄存器 **************************/
void MAX31865_Write(unsigned char addr, unsigned char data)
{
    unsigned char i;

    MAX31865_CS_CLR;
    
    // 发送地址（写操作）
    for(i = 0; i < 8; i++)
    {
        MAX31865_SCLK_CLR;
        if(addr & 0x80) MAX31865_SDI_SET;
        else MAX31865_SDI_CLR;
        MAX31865_SCLK_SET;
        addr <<= 1;
    }
    
    // 发送数据
    for(i = 0; i < 8; i++)
    {
        MAX31865_SCLK_CLR;
        if(data & 0x80) MAX31865_SDI_SET;
        else MAX31865_SDI_CLR;
        MAX31865_SCLK_SET;
        data <<= 1;
    }
    
    MAX31865_CS_SET;
}

/************************** 读寄存器 **************************/
unsigned char MAX31865_Read(unsigned char addr)
{
    unsigned char i;
    unsigned char data = 0;

    MAX31865_CS_CLR;
    
    // 发送地址（读操作）
    for(i = 0; i < 8; i++)
    {
        MAX31865_SCLK_CLR;
        if(addr & 0x80) MAX31865_SDI_SET;
        else MAX31865_SDI_CLR;
        MAX31865_SCLK_SET;
        addr <<= 1;
    }
    
    // 读取数据
    for(i = 0; i < 8; i++)
    {
        MAX31865_SCLK_CLR;
        data <<= 1;
        MAX31865_SCLK_SET;
        if(MAX31865_SDO_READ) data |= 0x01;
    }
    
    MAX31865_CS_SET;
    return data;
}

/************************** 获取温度 **************************/
float MAX31865_GetTemp(void)
{
    unsigned int data;
    float Rt;
    float Rt0 = 100.0f;
    float Z1, Z2, Z3, Z4, temp;
    float a = 3.9083e-3f;
    float b = -5.775e-7f;
    float rpoly;

    // 读取16位ADC数据
    data = MAX31865_Read(0x01) << 8;
    data |= MAX31865_Read(0x02);
    data >>= 1;  // 右移1位，取高15位

    // 计算PT100电阻值
    Rt = (float)data / 32768.0f * RREF;

    // Callendar-Van Dusen公式计算正温度
    Z1 = -a;
    Z2 = a * a - 4 * b;
    Z3 = 4 * b / Rt0;
    Z4 = 2 * b;
    temp = Z2 + Z3 * Rt;
    temp = (sqrt(temp) + Z1) / Z4;

    // 正温度直接返回
    if(temp >= 0.0f)
    {
        return temp;
    }

    // 负温度使用多项式拟合
    rpoly = Rt;
    temp = -242.02f;
    temp += 2.2228f * rpoly;
    rpoly *= Rt;
    temp += 2.5859e-3f * rpoly;
    rpoly *= Rt;
    temp -= 4.8260e-6f * rpoly;
    rpoly *= Rt;
    temp -= 2.8183e-8f * rpoly;
    rpoly *= Rt;
    temp += 1.5243e-10f * rpoly;

    return temp;
}
```

## 使用说明 {#使用说明}

### 1. 硬件连接步骤

1. **连接电源**：将MAX31865模块的VCC连接到3.3V，GND连接到GND
2. **连接SPI引脚**：按照接线表连接SDO、CS、SCLK、SDI到STM32
3. **连接PT100**：使用三线制方式连接PT100传感器
4. **检查接线**：确保所有连接牢固，无虚焊

### 2. 软件配置

1. **配置参考电阻**：根据模块实际参考电阻修改 `RTD_RREF_VALUE`
2. **选择滤波频率**：国内使用50Hz，国外使用60Hz
3. **选择接线方式**：根据实际接线修改 `RTD_WIRE_3LINE` 宏

### 3. 调试技巧

**常见问题及解决方法：**

| 问题现象 | 可能原因 | 解决方法 |
|---------|---------|---------|
| 温度显示固定值 | 通信故障 | 检查SPI接线、时钟频率 |
| 温度显示异常高/低 | 参考电阻配置错误 | 确认模块参考电阻值 |
| 显示RTD开路错误 | 传感器未连接或断线 | 检查PT100接线 |
| 显示RTD短路错误 | 传感器短路 | 检查PT100是否短路 |
| 温度跳变严重 | 电源干扰 | 添加电源滤波电容 |

### 4. 精度优化建议

- **使用三线制连接**：消除导线电阻影响
- **添加滤波电容**：在VCC和GND之间添加100nF电容
- **远离干扰源**：将传感器远离电机、继电器等干扰源
- **定期校准**：使用标准温度计进行校准

## 注意事项 {#注意事项}

1. **电源要求**：MAX31865需要稳定的3.3V电源，纹波应小于100mV
2. **参考电阻**：不同模块参考电阻可能不同，常见为400Ω或430Ω
3. **PT100类型**：确保使用的是PT100（100Ω）而非PT1000（1000Ω）
4. **测量范围**：标准PT100测量范围为-200°C至+850°C
5. **故障处理**：检测到故障时应及时处理，避免错误数据
